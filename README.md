# Install APK

## 简介
Install APK 是一款 macOS 批量安装 APK 工具，适合开发测试和快速版本迭代。

### 主要功能
- 支持桌面、下载夹及自定义目录
- 可批量安装多个 APK
- 安装失败可重试或进入命令行调试
- 提供安装结果总结与详细日志

## 版本历史

### v1.2 (2025.08.08)
- 新增自定义目录安装，支持输入路径或直接拖拽文件夹到终端
- 新增安装选项，可选择是否允许版本降级
- 错误解析增强，识别更多 adb 错误类型并提供解决提示
- 安装进度提示，显示 [当前 / 总数] 安装进度
- 日志统一存放到桌面 apk_install_logs，并带时间戳

### v1.0
